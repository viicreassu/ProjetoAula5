# Cadastro de Aluno - ASP.NET MVC

Projeto desenvolvido para demonstrar:

- Model Aluno
- Data Annotations
- Validação com ModelState
- Tag Helpers
- Formulário de Cadastro

## Tecnologias

- ASP.NET MVC
- C#
- Razor Views
- Bootstrap

## Testes

- Formulário vazio
- E-mail inválido
- Cadastro com dados válidos
